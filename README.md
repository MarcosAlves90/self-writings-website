# Estudo de Componentização com JavaScript Puro

Repositório de estudo focado em **componentização no front-end usando apenas JavaScript puro**, sem frameworks ou bibliotecas.  
A ideia é entender **como frameworks modernos funcionam internamente**, aplicando conceitos fundamentais com HTML, CSS e JS modernos.

## Objetivo
- Criar componentes reutilizáveis e organizados
- Entender encapsulamento, estado e renderização
- Estruturar interfaces de forma escalável sem abstrações

## Conceitos
- Componentização com JS puro
- Manipulação controlada do DOM
- Separação de responsabilidades
- Reutilização de código

## Tecnologias
- HTML5
- CSS3
- JavaScript (ES6+)
